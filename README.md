# Smart Facial Authentication System

## Project Overview
This project is a Smart Facial Authentication System developed using Python and OpenCV. It identifies and authenticates users through facial recognition technology.

## Features
- Face Detection
- Face Recognition
- User Authentication
- Secure Access Control

## Technologies Used
- Python
- OpenCV
- NumPy
- Machine Learning

## How to Run
1. Install Python
2. Install required libraries
3. Run the main Python file
4. Allow camera access
